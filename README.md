# Jamaica Hurricane Relief

This repository contains product documentation and code for the Jamaica Hurricane Relief project.

## Project Overview

This repository serves as the source of truth for product documentation, maintained as Markdown files, and will also contain the codebase for the relief distribution tracking system.

## Repository Structure

```
product-docs/
  ├── fieldtrack/
  │   ├── prd-fieldtrack.md
  │   └── technical-one-pager-fieldtrack.md
  └── general-context/
      ├── constraints.md
      ├── hurricane-context.md
      └── ticket-creation.md
```

## Documentation

All product documentation is maintained in Markdown format within the `product-docs/` directory.

## Contributing

Documentation and code changes should be committed with clear, descriptive commit messages.
